🔧 Clear Browser Cache

If you're seeing old versions of the app, click the buttons below to clear all cached data:

Quick Fix Steps:
  • Click "Clear All Caches" button above
  • Click "Unregister Service Workers" button
  • Click "Reload App" to open a fresh version
  • Or use Ctrl+Shift+R (Windows) / Cmd+Shift+R (Mac) for hard reload
Browser Settings (Alternative):
  • Chrome/Edge: Open DevTools (F12) → Application tab → Clear storage → Clear site data
  • Firefox: Open DevTools (F12) → Storage tab → Right-click → Clear All
  • Safari: Develop menu → Empty Caches
